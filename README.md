# Mobile App (Flutter)

This is a comprehensive mobile application built with Flutter, designed to manage content and user interactions effectively.

## Features

The application is divided into three main sections:

### 1. User Management
- **Admin**: Tools for administrators.
- **Learner**: Interface for learners.
- **Experts**: Dedicated section for experts.

### 2. Content Management
- **Lessons**: Manage and view educational lessons.
- **Music**: Music player and library features.
- **Podcasts**: Listen to and manage podcasts.
- **Videos**: Video content management.
- **Feedback**: Collect and view user feedback.

### 3. User Features
- **Track Progress**: Monitor user progress and achievements.
- **Subscription**: Manage user subscriptions.
- **Edit User**: Functionality to edit user profiles.

## Technologies Used
- **Flutter**: Cross-platform mobile development.
- **Firebase**: Backend services (Firestore, Core).

## Getting Started

To run this project locally:

1.  Ensure you have Flutter installed.
2.  Clone this repository.
3.  Run `flutter pub get` to install dependencies.
4.  Run `flutter run` to start the app.

> Note: Firebase configuration files (`google-services.json` and `GoogleService-Info.plist`) are required for the app to function correctly.
